# Netflix-Data-Visualization Project
## Overview
This project analyzes Netflix dataset to uncover insights about movies and TV shows.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Analysis
- Movies vs TV Shows trend
- Content by country
- Release year analysis
- Genre distribution

## Insights
- Netflix content increased rapidly after 2015
- Movies are more than TV Shows
- USA produces the most content
